¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se utiliza para añadir elementos interactivos a la página web.
Es un lenguaje de programación que se puede aplicar a un documento HTML. Con JavaScript puedes crear desde una simple Web hasta juegos y gráficos 3D.

¿Quién inventó JavaScript?

JavaScript, fue inventado por Brendan Eich en mayo de 1995, quien era el cofundador del proyecto Mozilla, Mozilla Foundation y la Corporación Mozilla.

Curiosidades de JavaScript

  1. Su nombre real no es JavaScript si no ECMAScript.
  2. La primera versión se lanzó en un periodo de 10 días.
  3. El primer nombre que recibió fué Mocha, posteriormente LiveScript, y tras unos meses, recibió el nombre de JavaScript.
  4. JavaScript es una marca registrada por Oracle.
  5. Ante la popularidad de JavaScript, Microsoft, decidió sacar su propia versión llamada JScript, su última versión estable es del 2011.

¿Qué es TypeScript?

TypeScript es un lenguaje de programación construido a un nivel superior que JS.
Esto quiere decir que Typescript dota al lenguaje de varias características adicionales que hacen que podamos escribir código en definitiva, más limpio y sólido.
Fue creado por microsoft en 2012, y Google lo adoptó como su lenguaje por defecto para desarrollar con Angular.
Como TypeScript es un superconjunto de JavaScript, todo el código escrito en JS es válido para TS, pero NO lo contrario.

Curiosidades de TypeScript

  1. Es apoyado por fabricantes, tanto Microsoft como Google apoyan de una forma fuerte este lenguaje.
  2. Es un lenguaje compilado por lo que podemos detectar muchos errores antes de abordar la ejecución.
  3. Se trata de un lenguaje moderno que permite la unión de types que permite que un tipo soporte 2 opciones. (string | number).

Diferencias entre JavaScript y TypeScript

Características JavaScript TypeScript
Tipados No existen aunque se pueden simular Lenguaje fuertemente tipado, donde se pueden crear tipos genéricos o interfaces
Modularización Através de ECMAScript 6 Soporte directo para módulos
Tuplas No las soporta Si las soporta
Orientación a objetos Más compleja Similar a otros lenguajes de programación como Java o C#
Interfaces No existe soporte para las interfaces Son imprescindibles

¿JavaScript o TypeScript?

  1. En proyectos pequeños, usar TypeScript es un poco brusco, por lo que quizá no merezca la pena.
  2. En proyectos medianos o grandes, escribir código en TS ofrece grandes ventajas que se van a notar a corto y a largo plazo.
  3. Si trabajas con frameworks, usar TypeScript es casi una obligación.
  4. A esto hay que sumarle que en todas las empresas de desarrollo software los principales proyectos utilizan alguno de estos frameworks o librerías.
  5. Lo más recomendable en caso de duda es aprender TypeScript, ya que a nivel profesional es un lenguaje más útil.
