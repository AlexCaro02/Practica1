¿Qué es GitHub?

GitHub es una plataforma en línea que ofrece servicios de alojamiento y control de versiones para proyectos de software basada en Git.

Esta fue diseñada por Github Inc. en 2008 usando Ruby on Rails y fue comprada por Microsoft en 2018.

Es considerada la plataforma más grande de colaboración para proyectos de código abierto.

¿Qué es un "Git"?

Git es software de código abierto de control de versiones diseñado por Linus Torvalds, creador de Linux, con la meta de mejorar la eficiencia, confiabilidad y compatibilidad del mantenimiento de versiones de grandes aplicaciones, con características tales como:

  1. Es un software gratuito.
  2. No depende de un repo central.
  3. Podemos mantener un listado completo del historial de versiones.
  4. Podemos acceder a estas versiones fácilmente así como revertirlas.
  5. Un sistema de ramificación de versiones diferentes de la rama main.
  6. Posibilidad de unir esas ramas a la rama main.

Este consiste en llevar un registro de los cambios en archivos incluyendo coordinar el trabajo de varias personas sobre estos.

Para coordingar el trabajo de un grupo de personas en torno a un proyecto, se realizan unos acuerdos de como se va a trabajar con Git (el flujo de trabajo).
En GitHub, se utiliza un flujo llamado GitHub-Flow, en el que se dividen las versiones en ramas mediante estas reglas:

  1. Master: Rama de versiones listas para ser puestas en producción.
  2. Cualquier rama creada a partir de master tendrá su propio nombre escrito por el usuario.
  3. Cuando crees que tu rama esta lista para unirse a master, se realiza un pull request.
  4. Tras ser revisado por otra persona, esta se unirá al master.

Otras herramientas de Git

Existen diversos softwares que utilizan Git como base aparte de GitHub.

Los principales competidores, es decir, otras plataformas de forja, son GitLab y BitBucket, pero su mayor desventaja frente a este son las limitaciones detras de pago de estas plataformas.
